About IngrediList

IngrediList is an AI-powered iOS app that turns physical or digital recipes into organised shopping lists synced with Apple Reminders. Built with privacy in mind, it runs entirely on-device using Apple's Foundation Models framework.

Whether you're scanning a cookbook, a screenshot, or a handwritten recipe card, IngrediList uses advanced OCR and natural language processing to instantly extract ingredients, identify quantities, and create a perfectly formatted shopping list—all without sending your data to the cloud.

Key Features:

  • Smart OCR for scanning recipes from any source
  • On-device AI processing for complete privacy
  • One-tap sync with Apple Reminders
  • Store cupboard tracking to avoid duplicate purchases
  • Automatic serving size scaling
  • Multi-recipe merge for menu planning
  • Beautiful, native iOS design with Liquid Glass aesthetic

Brand Guidelines

Logo Usage

The IngrediList logo should always appear with adequate clear space and should not be distorted, rotated, or recoloured. When used on light backgrounds, use the primary colour version. On dark backgrounds, use the white version.

Typography

IngrediList uses SF Pro Display (or Inter as a web fallback) for all text. Headings use weights 600–700, body text uses weight 400.

Tone of Voice

IngrediList's communication is calm, smart, and helpful. We use British English spelling and write in a friendly yet professional tone. We prioritise clarity and avoid jargon.
